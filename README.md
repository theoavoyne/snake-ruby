# Snakes Game

Ruby Snakes game.

## Getting Started

### Prerequisites

```
Ruby 2.4.3
```

### Installing

Clone the repository on your local machine, move inside the repository and run this command:

```
ruby app.rb
```

## Built with

* [Ruby](https://www.ruby-lang.org/fr/)
* [GOSU](https://www.libgosu.org/) -  Game development library for Ruby
