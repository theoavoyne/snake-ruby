# Snakes Game

Ruby Snakes game.

## Getting Started

### Prerequisites

Ruby 2.4.3

### Installing

Clone the repository on your local machine and run:

```
ruby snake-ruby
```

## Built with

* [Ruby](https://www.ruby-lang.org/fr/)
* [GOSU](https://www.libgosu.org/) -  Game development library for Ruby
